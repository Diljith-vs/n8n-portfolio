# Leave Approval System

## Overview
This workflow automates a leave approval process.

## Features
- Takes employee details
- Checks leave eligibility
- Approves or rejects requests
- Generates a personalized message

## Nodes Used
- Manual Trigger
- Edit Fields
- IF
- Edit Fields (Approved)
- Edit Fields (Rejected)

## Skills Demonstrated
- Conditional Logic
- Expressions
- Data Transformation
- Workflow Automation
