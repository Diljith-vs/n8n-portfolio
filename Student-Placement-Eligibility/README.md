# Student Placement Eligibility System

## 📌 Description
This n8n workflow automates student placement eligibility based on CGPA.

## 🚀 Workflow
1. Accept student details.
2. Check if CGPA is greater than or equal to 7.
3. Generate placement status.
4. Create a personalized message.

## 🛠 Technologies
- n8n
- IF Node
- Edit Fields
- Expressions
- Workflow Automation

## 📂 Files
- `student-placement-eligibility.json`
- `student-placement-workflow.png`

## 📖 Example Output

```
Hello Diljith,

Branch: CSE

CGPA: 8

Status: Eligible
```
