# Student Placement Eligibility System

## Description
This workflow automates placement eligibility based on student CGPA.

## Workflow
1. Accepts student details.
2. Checks if CGPA is greater than or equal to 7.
3. Generates placement status.
4. Creates a personalized message.

## Skills Used
- n8n
- Workflow Automation
- IF Node
- Expressions
- Business Logic
